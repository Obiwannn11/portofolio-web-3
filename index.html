About Me

Sebagai mahasiswa Teknik Informatika semester akhir di UIN Alauddin Makassar, saya fokus pada pengembangan backend systems yang robust dan scalable. Perjalanan saya dimulai dari ketertarikan mendalam terhadap bagaimana data flows dan system architecture bekerja di balik layar.

Dengan pengalaman membangun berbagai website dan API systems, saya telah berkontribusi dalam proyek-proyek seperti sistem akademik kampus, e-commerce backend, dan company profile websites. Saat ini sedang menyelesaikan skripsi tentang RAG (Retrieval-Augmented Generation), mengeksplorasi intersection antara backend development dan AI technologies.

Saya percaya bahwa good code adalah kombinasi antara clean architecture, efficient performance, dan maintainability. When I'm not coding, saya suka vibecoding sambil eksplorasi teknologi baru dan best practices dalam software development.
